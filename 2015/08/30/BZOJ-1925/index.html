BZOJ 1925:[Sdoi2010]地精部落

Description
一座长度为 N 的山脉 H可分 为从左到右的 N 段，每段有一个独一无二的高度 Hi，其中Hi是1到N 之间的正 整数。 如果一段山脉比所有与它相邻的山脉都高，则这段山脉是一个山峰。位于边 缘的山脉只有一段相邻的山脉，其他都有两段（即左边和右边）。 类似地，如果一段山脉比所有它相邻的山脉都低，则这段山脉是一个山谷。酒馆可以设立在山谷之中。每座山峰上都可以设立瞭望台。 地精们希望这N 段山脉每段都可以修建瞭望台或酒馆的其中之一，只有满足 这个条件的整座山脉才可能有地精居住。 现在你希望知道，长度为N 的可能有地精居住的山脉有多少种。两座山脉A 和B不同当且仅当存在一个 i，使得 Ai≠Bi。由于这个数目可能很大，你只对它 除以P的余数感兴趣。

无比的难想，但是想通之后特别好写ORZ..
定义f[i][j]为1~i第一个数是j且第一段下降的种类数，g[i][j]为第一段上升的种类数，在【1，i】的区间里，f和g显然是对称的，f[i][j]=g[i][i-j+1],所以g数组不用开出，需要用到的时候直接用f表示。
对于f[i][j],分第二位是j-1和小于j-1两种。

①第二位小于j-1：这时的种类数为f[i][j-1]。相当于交换j-1与j，j的左右不会有正好比它小一的数，所以交换后仍然成立。
②第二位为j-1：第二位开始为以j-1为第一个树的第一段上升的序列g[i-1][j],即f[i-1][i-j+1]。
